\begin{itemize}[

    \item \textbf{iToBoS Annotation Tool:} Developed a desktop application using React, Electron, and Vite for efficient management of YOLO-format annotations. Increased annotation efficiency by 40\% and reduced error rates by 25\%, supporting the EU-funded melanoma detection project. \href{https://github.com/adeola-jo/iToBoS-Annotation-Tool}{GitHub}

    \item \textbf{Camera Calibration:} Implemented a robust camera calibration system using OpenCV and Python to extract intrinsic and extrinsic parameters from multi-view images. Achieved sub-millimeter accuracy in 3D reconstruction tasks for robotic vision applications. \href{https://github.com/adeola-jo/CameraCalibration/}{GitHub}

    \item \textbf{Stereo Visual Odometry:} Created a real-time stereo visual odometry system for the KITTI dataset using feature detection, tracking, and 3D reconstruction. Achieved 96.5\% accuracy in trajectory estimation with optimized C++ implementation. \href{https://github.com/AdeolaJoseph/Stereo-Visual-Odometry}{GitHub}

    \item \textbf{Privacy-Preserving Federated Learning:} Designed a framework combining federated learning with differential privacy techniques, enabling collaborative model training while preserving data privacy. Implemented non-IID data handling and model compression, demonstrating only 2.8\% accuracy loss compared to centralized learning. \href{https://github.com/adeola-jo/PrivacyFederatedLearning}{GitHub}
    
    \item \textbf{Skin Lesion Detection:} Implemented and compared multiple deep learning architectures (CNNs, Vision Transformers) for melanoma detection, achieving 89.7\% sensitivity and 85.3\% specificity, exceeding published benchmarks by 7\%. Developed custom data augmentation techniques to address class imbalance. \href{https://github.com/adeola-jo/Pigmented-Skin-Lesion-Detection-in-Clinical-Images-Using-Deep-Learning-Methodologies}{GitHub}
    
    \item \textbf{Contrastive Metric Embedding:} Developed an unsupervised similarity learning framework using contrastive loss on the MNIST dataset. Implemented in PyTorch with custom embedding techniques, achieving 93.2\% clustering accuracy without labeled data. \href{https://github.com/adeola-jo/Metric-Embedding-Similarity-Learning}{GitHub}

    \item \textbf{Reinforcement Learning for Robot Motion Planning:} Implemented Q-Learning algorithms for autonomous navigation in complex environments. Developed a custom reward function that reduced path planning time by 35\% compared to traditional methods, with successful validation in simulated obstacle courses. \href{https://github.com/adeola-jo/Reinforcement-Learning}{GitHub}

    \item \textbf{TabiWilson Photography Website:} Designed and developed a responsive portfolio website using HTML5, CSS3, JavaScript, and Bootstrap. Implemented custom image galleries, contact forms, and SEO optimization, resulting in a 40\% increase in client inquiries for the photography business. \href{https://github.com/adeola-jo/TabiWilson-Photography/}{GitHub}
    
    \item \textbf{CNN-Based Image Classification:} Developed and compared various CNN architectures (ResNet, VGG, DenseNet) for image classification on the CIFAR dataset. Achieved 92.3\% accuracy through transfer learning and custom data augmentation techniques, with implementation in PyTorch. \href{https://github.com/adeola-jo/CNN-Image-Classification}{GitHub}

    \item \textbf{Multi-Robot Swarm Control:} Implemented Reynolds' flocking rules for coordinating robot swarms in ROS/Gazebo. Developed custom collision avoidance algorithms and collective decision-making behaviors that enabled effective navigation in cluttered environments with up to 20 robots. \href{https://adeola-jo.github.io/multirobotswarm/}{GitHub}

    \item \textbf{Multi-Modal Human-Swarm Interaction:} Created an innovative interface combining vision, speech recognition, and multi-agent LLMs to enable natural human interaction with robot swarms. Implemented in ROS with Python, achieving 92\% command recognition accuracy and intuitive control of heterogeneous robot teams. \href{https://github.com/adeola-jo/human-swarm-interaction}{GitHub}
    
    \item \textbf{Pose-Based EKF SLAM:} Developed a real-time SLAM system combining EKF with ICP scan-matching for Kobuki Turtlebot. Achieved centimeter-level mapping accuracy in indoor environments through optimized feature extraction and data association algorithms. \href{https://github.com/adeola-jo/Pose-Based-EKF-SLAM-ICP}{GitHub}

    \item \textbf{Extended Kalman Filter (EKF) Map-Based Localization:} Implemented an EKF-based localization system using line features. The system provides position estimates based on sensor input. \href{https://github.com/adeola-jo/Line-Feature-EKF-Map-Localization}{GitHub} | \href{https://github.com/adeola-jo/Line-Feature-EKF-Map-Localization/blob/main/report.pdf}{PDF}
